📊 E-Commerce Sales Analysis Dashboard (Excel)

🚀 Project Overview

This project is an E-Commerce Sales Analysis Dashboard built entirely using Microsoft Excel. It provides key business insights such as sales performance, profit trends, category-wise analysis, and regional distribution using interactive visuals and filters.

The dashboard helps stakeholders quickly understand:

Sales & profit trends
Category performance
Regional sales distribution
Customer segments & yearly performance

🔍 Key Excel Techniques Used

1. KPI Calculations
Total Sales = SUM(Sales)

Total Profit = SUM(Profit)

Total Orders = COUNT(Order ID)

Profit Margin = (Total Profit / Total Sales) * 100
2. Year-over-Year (YOY) Growth
YOY Growth % = 
((Current Year Value - Previous Year Value) / Previous Year Value) * 100
3. Pivot Tables

Used for:

Monthly Sales & Profit trends
Category-wise Profit
Top Sub-categories
Region-based filtering
4. Top 10 Subcategories
Applied sorting in Pivot Table
Used Top N Filter:
Value Filter → Top 10 items by Sales
5. Map Visualization
Used Filled Map Chart
Based on State & Sales data
6. Slicers (Filters)
Region
Segment
Year

Connected slicers to multiple Pivot Tables for interactivity.
